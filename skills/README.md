# Prometheus Agent Skills

This directory contains local skills for the Prometheus inspection agent.

- `promql-authoring`: PromQL writing rules and common query examples.
- `metric-domain-knowledge`: Core metric selection and domain risk knowledge.
- `inspection-workflow`: Interactive and scheduled inspection orchestration rules.

Use these skills as AI-platform prompt resources. Deterministic operations such as Prometheus querying, Grafana threshold loading, metric analysis, severity grading, and HTML rendering should remain Python tools.
